# C++ 植物大战僵尸塔防游戏复刻

## 项目概述

本项目基于 C++ 复刻经典塔防游戏《植物大战僵尸》的核心玩法，目标是通过完整的工程实现，系统训练面向对象设计、游戏主循环、碰撞检测与对象生命周期管理能力。

## 核心功能

- 植物种植与卡槽放置逻辑
- 僵尸波次生成、移动与攻击行为
- 植物自动射击、子弹飞行与命中判定
- 游戏胜负判断与基础状态管理

## 技术实现

### 1) 架构设计

- 以统一游戏实体基类管理植物、僵尸、子弹等对象
- 使用多态分发实体更新与渲染逻辑，降低模块耦合

### 2) 主循环与时间步长

- 使用基于 `deltaTime` 的固定步长主循环
- 降低不同设备性能差异带来的游戏速度偏差

### 3) 碰撞检测

- 实现 AABB（Axis-Aligned Bounding Box）碰撞检测
- 完成子弹-僵尸、僵尸-植物等核心碰撞场景

### 4) 内存与对象管理

- 基于 STL 容器维护动态实体集合
- 在生命周期结束时清理失效对象，避免内存泄漏风险

## 项目亮点

- 代码量约 1500 行，包含完整塔防核心流程
- 在 Windows 平台稳定运行，帧率可保持在 60 FPS 左右
- 具备较好的可扩展性，便于后续新增植物/僵尸类型
